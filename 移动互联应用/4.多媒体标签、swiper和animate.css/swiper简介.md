# swiper简介

官网地址：<https://swiperjs.com/>

参考地址：<https://www.swiper.com.cn/>

## swiper是什么
swiper 是个专门用于播放滚动动画的工具

## 作用
用于制作滚动播放的播放器

## 安装

### cdn

从 <https://swiperjs.com/get-started#use-swiper-from-cdn> 找到 `cdn` 地址并引入即可

### npm or yarn

```shell
npm i swiper
```

## 使用的思路

1. swiper 安装好以后，进行布局
2. 初始化一个 swiper 对象
3. 按需要添加 swiper 模块
