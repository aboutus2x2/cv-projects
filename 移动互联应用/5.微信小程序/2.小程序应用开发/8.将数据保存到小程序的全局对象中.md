# 将数据保存到小程序的全局对象中

```js
// 获取小程序实例对象
const app = getApp()

app.globalData.x = 1 // 给小程序的全局对象赋值

// 只要不关闭小程序就能通过小程序全局对象保存参数
```

## 为什么要使用全局对象来存储数据

为了让每个页面共享相同的数据

## 持久化数据

什么叫持久化？持久化就是长期存储数据到磁盘中，关闭应用程序后，存储的数据还在

异步是什么？异步就是程序执行的时候不会等待或卡顿

同步是什么？同步的程序在调用的时候会等待或卡顿
